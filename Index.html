Control FM4 Java y SQL Este proyecto consistió en desarrollar una interfaz gráfica en Java conectada a una base de datos, para permitir registrar la informacion de los migrantes al llegar (nombre, fecha de nacimiento, cuanto tiempo se iban a quedar, que materiales se les entregaban, etc). También permitía generar un reporte en pdf con la información registrada en la base datos dado un determinado rango de tiempo.
Ingesta de calorías Python Este programa pide los datos del usuario (género, edad, peso, estatura en cm y la cantidad de ejercicio que realiza) para posteriormente calcular la tasa metabólica basal y se multiplica por la cantidad de actividad física que realiza, siendo el resultado la cantidad de calorias que debe consumir al dia. Despues se le pregunta si desea mantener su peso, reducirlo o aumentar, en base a su elección, se le suman 500 calorías al resultado anterior si desea subir de peso, se restan 500 calorías si desea bajar, y se mantiene si desea mantener su peso actual. Después, con un algoritmo genético binario y distintos menús guardados en arreglos, se obtenian los alimentos que se podian consumir en el desayuno, comida y cena, sin superar la cantidad maxima de calorias que se habia obtenido anteriormente.
Juego 21 C++ Siendo uno de los primeros programas que realice, este consistia en implementar el juego 21 de forma libre, usando c++ sin interfaz grafica, siendo el usuario el jugador contra la maquina. Se le pedia el nombre al usuario, y posteriormente iniciaba el juego, mostrando la carta que le habia salido en cosola (dibujada con código ascii), terminaba una vez que el usuario o la maquina llegaba a 21, o en caso de retirarse, el que quedara mas cerca de esa cantidad.
